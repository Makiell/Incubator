settings.py - константы и настройки для игры

tetromino.py - файл отвечающий за спавнящийся блок

tetris.py - весь функционал и текст игры

main.py - файл загрузок и запуска игры

scores - папка с файлами для очков игр

last.txt - файл для хранения очков последней игры

record.txt - файл для хранения данных о рекорде

assets - папка с картинками и шрифтом

При запуске файла не забудьте поменять в коде путь к картинкам, файлам и шрифту
